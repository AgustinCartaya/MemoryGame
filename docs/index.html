MemoryGame

Este proyecto fue creado como peticion del modulo Info2B de la Universidad de Bourgogna para poner en practica los conocimientos aprendidos en java utilizando la libreria java swing.

Objetivo Principal

Desarrollar un juego de memoria, con algunas funcionalidades extra en java

Objetivos Específicos

      Crear la interface principal del juego
      Estructurar las clases y dependencias del juego
      Agregar las funcionalidades extra

Recursos usados

      Lenguaje de programación: JAVA, versión 1.8.0_241
      IDE de desarrollo: NET BEANS 8.2
